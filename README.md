Previewing locally (Windows)
----------------------------
Option A — Open directly in your browser (double-click or run from PowerShell):

```powershell
Start-Process .\index.html
```

Option B — Run a lightweight local server (recommended for testing relative paths):

```powershell
# If you have Python 3 installed
python -m http.server 8000
# Then open http://localhost:8000 in your browser
```
